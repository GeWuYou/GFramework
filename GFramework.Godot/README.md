# GFramework.Godot

GFramework 框架的 Godot 引擎集成模块，提供Godot特定的功能和扩展。

## 主要功能

- **Extensions** - Godot节点扩展方法，简化常见开发任务
- **Signal** - 流畅的信号连接API，支持链式调用
- **Storage** - Godot文件存储系统，支持虚拟路径
- **Settings** - Godot设置系统，管理音频和图形设置

## 依赖关系

- 依赖 GFramework.Core
- 依赖 GFramework.Core.Abstractions
